* 권장: Google Chrome 버젼 77 이상

## 기술 스택 (Technique Used) (예시)
### Server(back-end)
 - golang 1.15.2
 - GoCV (OpenCV Binding) 
 - mongoDB
 - Express
 
### front-end
 - React.js  
 - Antd UI Framework  
 - Socket.io

$ yarn start or npm run start
